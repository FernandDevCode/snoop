Centre d'Aide

Votre ID de connexion unique est affiché en grand sur la page d'Accueil, dans la carte bleue intitulée "Votre ID". C'est cet identifiant que vous devez communiquer à la personne qui souhaite se connecter à votre ordinateur.

Sur la page d'Accueil, entrez l'ID de l'ordinateur de votre partenaire dans le champ "Identifiant du pc" situé dans la section "Accédez à un pc distant". Cliquez ensuite sur le bouton "Se connecter". Votre partenaire recevra une notification pour accepter ou refuser votre demande de connexion.

Si une connexion ne peut être établie, veuillez vérifier les points suivants :
  • Assurez-vous que l'ID que vous avez saisi est correct.
  • Vérifiez que vous et votre partenaire êtes bien connectés à Internet.
  • Assurez-vous que l'application SNOOP est bien lancée sur l'ordinateur de votre partenaire.
